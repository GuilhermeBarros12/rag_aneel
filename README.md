# Extrator de Dados ANEEL - Projeto RAG

Este repositório contém os scripts iniciais de Engenharia de Dados para o pipeline do nosso RAG.

## Como rodar na sua máquina:

1. Clone este repositório.
2. Instale as dependências executando no terminal:
   `pip install -r requirements.txt`
3. Rode os scripts de teste:
   - `python entender_json.py` (Para mapear metadados)
   - `python extrator_pdf.py` (Para gerar o Markdown das tabelas)
